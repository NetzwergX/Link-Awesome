Link-Awesome

Link-Awesome 1.0

Awesome visual cues (icons) for your hyperlinks.

Let your visitors see where you want to take them at the first glance!
Link-Awesome adds small icons to your external links, it supports all major social networking sites & video portals, as well as file type hinting for images, downloads and documents.

What's new?

Link-Awesome 1.0 is the first "stable" release of Link-Awesome. It now uses Font-Awesome 4.0 and has become even better "under the hood". And all this while staying backwards-compatible with Link-Awesome 0.1!

Improved Shipping

I now ship a compiled CSS file as well as a minified version if said CSS file in addition to the original LESS file. This makes integration Link-Awesome into your site a breeze.

New Icon

Since Font-Awesome 4.0 now includes an explicit stackexchange icon, the number of social icons has gone up to 22, supporting 24 different places on the web.

Getting Link-Awesome

Getting Link-Awesome was never easier. As of now, you still need to include Font-Awesome yourself, but I am working on providing a full version that already includes Font-Awesome in the future.

If you already have Font Awesome included, head over to GitHub and download the link-awesome.min.css and include it in the head of your document: 

<link rel="stylesheet" href="link-awesome.min.css" />
